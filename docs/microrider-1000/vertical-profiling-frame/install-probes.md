# Install the Microstructure Probes

Replace the factory-installed test probes with the deployment shear and FP07 temperature probes.

!!! warning "Fragile probes"

    The shear probes and FP07 temperature probes are extremely fragile and can be permanently damaged by contact with fingers, clothing or any solid object. Handle probes only by the metal body and keep the protective sleeves fitted until immediately before installation.

!!! note "When to install"

    Install the deployment probes immediately before deployment to minimise the risk of accidental damage during handling.

---

## Procedure

### 1. Remove the Test Probe

<photo>

1. Loosen the probe holder cap using the probe installation tool (approximately **½–¾ turn**).
2. Carefully withdraw the test probe until the SMC connector is accessible.
3. Hold the SMC connector body and disconnect the test probe.
4. Leave the SMC cable protruding from the probe holder.

---

### 2. Assemble the Deployment Probe

<photo>

1. Verify the ferrule and O-ring are installed as shown in **Figure&nbsp;1**.
2. Remove the protective sleeve from the deployment probe.
3. Connect the SMC connector to the probe.
4. Confirm the SMC connector is fully engaged.

!!! important

    Never pull on the probe cable. Always hold the SMC connector body when connecting or disconnecting a probe.

---

### 3. Install the Deployment Probe

<photo>

1. Carefully insert the probe into the probe holder.
2. Seat the probe fully into the holder.
3. Finger-tighten the probe holder cap.
4. Tighten the cap using the probe installation tool by an additional **⅛–¼ turn**.

!!! important

    The probe should seat with minimal force. If resistance is encountered, remove the probe, inspect the alignment and try again. Never force the probe into the holder.

---

### 4. Align the Shear Probe

<photo>

1. Rotate the shear probe to the correct orientation.
2. Hold the probe in position while tightening the probe holder cap.
3. Verify the probe orientation matches **Figure&nbsp;2**.

<figure markdown>

![Shear probe orientation](images/shear_probe_orientation.jpg){ width="70%" }

<figcaption>

**Figure 2.** Correct orientation of the installed shear probe.

</figcaption>

</figure>

!!! important

    Only the shear probes require orientation. The FP07 temperature probes may be installed in any rotational orientation.

---

### 5. Repeat for Remaining Probes

Repeat **Steps 1–4** for the remaining probes until all four deployment probes have been installed.

---

### 6. Final Inspection

<photo>

1. Verify all probe holder caps are tightened.
2. Verify each probe is fully seated.
3. Inspect all probe tips for visible damage.
4. Remove all protective sleeves before deployment.
